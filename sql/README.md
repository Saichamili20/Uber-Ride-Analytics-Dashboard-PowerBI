# SQL Analysis – Uber Ride Analytics Dashboard

## Overview
This folder contains SQL queries used for data cleaning, transformation, and business analysis of Uber ride data. The queries help extract meaningful insights such as ride patterns, revenue trends, cancellations, customer behavior, and performance of different vehicle types.

---

## Objective of SQL Analysis
The main objectives of using SQL in this project are:

- Clean and prepare raw Uber ride data
- Handle missing values and duplicates
- Analyze booking trends and ride behavior
- Calculate key business metrics (KPIs)
- Support Power BI dashboard development

---

## Key SQL Operations Performed

### 1. Data Cleaning
- Removed duplicate records
- Handled NULL values in important fields such as:
  - Booking Value
  - Ride Distance
  - Customer Rating
- Standardized inconsistent data formats

---

### 2. Exploratory Data Analysis (EDA)
SQL was used to understand:
- Total number of bookings
- Completed vs cancelled rides
- Distribution of vehicle types
- Payment method preferences

---

### 3. Revenue Analysis
- Calculated total revenue generated by each vehicle type
- Identified highest revenue-generating categories
- Compared average booking value across ride types

---

### 4. Ride Completion Analysis
- Measured completed vs cancelled ride percentage
- Identified cancellation patterns

---
  
### 5. Customer Behavior Analysis
- Most preferred payment method
- Average customer rating
- Peak demand patterns

---

### 6. Cancellation Analysis
- Customer cancellation reasons
- Driver cancellation patterns
- Identified major operational issues

---

### 7. Location & Distance Insights
- Most popular pickup locations
= Longest and average trip distances
- High-demand regions

---

### Key Metrics Derived Using SQL
- Total Bookings
- Completion Rate
- Cancellation Rate
- Total Revenue
- Average Booking Value
- Average Customer Rating

---

### Outcome

This SQL layer acts as the foundation of the project by transforming raw Uber ride data into structured, analysis-ready information used for business decision-making.
